COMPUTER CARE ACADEMY — EXCEL IMPORT MASTER GUIDE
===================================================

HOW TO IMPORT DATA:
1. Open any page (Students, Reports, etc.)
2. Click the green "📥 Import Excel" button
3. Drop your .xlsx file or browse for it
4. Wait for the import to complete

SHEET NAMES (must be exact):
- Students
- Teachers
- Fees
- Salary
- Topics
- Tasks
- Internships
- Courses
- Notifications

Qualification | Religion | Caste | Marital Status | Family Members | Annual Income

Admission Date | Status | Username | Password | Grade | Percentage | Rank | Reference

MATCHING RULES:
- Students matched by Mobile number (updates existing, adds new)
- Teachers matched by Name or Mobile (updates existing, adds new)
- Fees/Salary: always appended as new payments
- Topics/Tasks/Internships: always added as new records

DATE FORMAT: YYYY-MM-DD (e.g. 2024-06-15)
COURSE STATUS VALUES: Pending | Continue | Complete | Leave | Left
